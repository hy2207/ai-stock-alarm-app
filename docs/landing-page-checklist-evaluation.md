# AI Stock Alarm 랜딩페이지 최종 평가 문서

평가 대상: `src/app/pages/LandingPageV2.tsx`

평가일: 2026-04-29

## Executive Summary

현재 랜딩페이지는 **C 유형, 결과 지향형 자동화 솔루션**에 가장 가깝다. 사용자가 복잡한 시장 리서치를 직접 수행하는 대신, 관심종목을 입력하면 오늘의 행동 카드가 나온다는 구조가 중심이기 때문이다.

핵심 코어 필수요소는 전반적으로 충족한다. 특히 헤드라인, CTA 반복 배치, Input-Output 구조, 결과 카드 샘플, 리서치 시간 절감 비교는 명확하다.

다만 신뢰 증거는 아직 “시장 데이터 소스 예시” 수준이다. 실제 사용자 수, 누적 생성 카드 수, 베타 사용자 리뷰, 데이터 공급원 고지 등으로 보강하면 전환 설득력이 더 커진다.

최종 판정: **출시 가능한 초안이며, 신뢰 증거 보강 시 전환형 랜딩페이지로 강화 가능**

- 랜딩페이지의 핵심 문구가 “미국주식 판단을 아침 3분 안에 끝내세요”, “종목을 넣으면, 오늘의 행동 카드가 나옵니다”로 구성되어 있다.
- CTA도 “무료 데모로 시작하기”, “내 관심종목으로 확인하기”, “지금 카드 확인하기”처럼 결과 체험으로 유도한다.

## 우선순위 개선안

1. 신뢰 증거 강화

- “누적 생성 카드 수”, “베타 사용자 수”, “재방문율”, “알림 클릭률” 중 하나를 실제 지표로 추가한다.
- 실제 데이터가 아직 없다면 “베타 테스트 중”, “샘플 카드 기준”처럼 과장 없이 표시한다.

2. 히어로 CTA 문구 통일

- 현재 CTA가 여러 문구로 분산되어 있다.
- 기본 CTA를 “오늘 카드 확인하기”로 통일하고, 보조 CTA만 “로그인하고 설정하기”로 유지한다.

3. 결과 갤러리 차별화

- 같은 종목이 안정형/중립형/공격형에서 어떻게 달라지는지 보여주는 비교 카드를 추가한다.
- 이 기능은 AI Stock Alarm의 “개인화된 리스크 기준”을 더 잘 설명한다.

4. ROI 근거 보강

- “45-90분”의 근거를 단계별로 분해한다.
- 예: 뉴스 확인 15분, 실적 일정 확인 10분, 가격대 계산 20분, 손절 기준 설정 10분.

5. 투자 서비스 신뢰/면책 강화

- “투자 자문이 아닌 참고용 리서치” 문구는 이미 존재한다.
- 추가로 “추천 생성 기준”, “데이터 지연 가능성”, “사용자 최종 판단 원칙”을 짧게 고지하면 불안을 줄일 수 있다.
