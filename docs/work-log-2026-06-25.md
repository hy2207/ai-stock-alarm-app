### 9. `thinkingLevel` → `thinkingBudget` 수정

**파일**: `src/lib/llm/generateRecommendationCards.ts`

**원인**: `gemini-2.5-flash`는 `thinkingLevel` 파라미터를 지원하지 않음.
API 오류: `"Thinking level is not supported for this model."`

**수정**:
```typescript
// 변경 전
thinkingConfig: { thinkingLevel: "minimal", includeThoughts: false }

// 변경 후
thinkingConfig: { thinkingBudget: 0 }
```

`thinkingBudget: 0` = thinking 토큰 예산 0 (최소 thinking, 동일한 효과)

**커밋**: `e8cc166`
